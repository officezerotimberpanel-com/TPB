# 木の力で未来をまもり・つくる

## ティンバーパネル事業部

私たちは、地産地消を促進し河川で起こる洗堀を防ぐ根本を見直すと同時に多自然化を目標とし底生成物へのアプローチを狙う事で水生生物等から始まる食物連鎖を見直していきます。

間伐材、地域材等の小径木の流通を確立し確かな技術と品質管理のもとで製造されたティンバーパネルは、河床工、根固め工、木工沈床工、堰堤、砂防等の幅広い工事で貢献しております。

### ティンバーパネルの特長

* 洗堀を防ぐ構造
* 環境に配慮した木質（間伐材）建材
* 特許を持つ施工性と加工性
* 温もりあるデザイン性
* 底生生物へのアプローチを行う事で食物連鎖の回復、拡大を実現
* 旧ＮＥＴＩＳ番号取得済み

### 製品ラインナップ

※共通【多自然化対応・地産地消】

#### ティンバーパネル（TypeNatural）

河床工や根固め工をはじめとする河川構造物に対応した標準仕様のティンバーパネルです。栗石、玉石を使用する事で生物生息環境の創出を両立します。

#### ティンバーパネルブロック（Type***Natural/C）***

従来の木工沈床では対応をしきれない流速、流量に対して中身をコンクリートを用いる事で河川改修や災害復旧等の幅広い施工に対応。

#### ティンバーパネルブロック（CustomDesign）

現場条件や設計条件に合わせた自由設計に対応しております。サイズ・形状・使用材など、お客様のご要望に応じて製作いたします。※法面カバー工法

**ティンバーパネルブロック（NaturalPorous）**

通常製品の天端部分に多数の空隙を持つ多孔質コンクリートで、透水性や保水性に優れ、環境に配慮した一層を用いた環境配慮特化型。

**ティンバーパネルブロック（SuppressBank）**

通常製品とは異なり落差（上下）を付ける抑制配慮型。

**ティンバーパネルブロック（Planted Stone）**

通常製品の天端に現地石等を用いた景観配慮型。

**ティンバーパネルブロック（TypeZERO）**

既存するプレキャスト（製品として撤去を予定している製品等）をリユースする事で環境配慮、景観配慮、洗堀配慮等の多方面に対して現在注目をされているモデルとなります。

### 製品の強み

#### 1. 洗堀対策と環境配慮を両立

従来の護岸・根固め工では難しかった「洗堀防止」と「生物生息環境の創出」を同時に実現。河川構造物としての機能を確保しながら、多自然川づくりに貢献します。

#### 2. 地域材・間伐材の有効活用

地域で発生する間伐材や小径木を積極的に活用することで、森林資源の循環利用を促進。地産地消による輸送負荷の低減と地域経済の活性化に寄与します。

#### 3. 底生生物の生息環境を創出

パネル内部や周辺に形成される空隙が底生生物の生息場所となり、水生昆虫・甲殻類・魚類へとつながる食物連鎖の回復を支援します。

#### 4. 高い施工性と柔軟な設計対応

特許技術を活用した構造により施工効率を向上。現場条件や流況に応じた自由設計にも対応し、新設工事から災害復旧工事まで幅広く活用できます。

#### 5. 景観との調和

木材ならではの自然な風合いにより、周辺環境との調和を実現。コンクリート構造物では得られない温かみのある景観形成に貢献します。

#### 6. 持続可能なインフラ整備への貢献

河川・砂防・農業土木分野において、防災・減災機能と生物多様性保全を両立する次世代型の環境配慮製品として、多くの現場で採用が検討されています。
